# Signal Prediction System using LSTM

An interactive time-series signal prediction system built using Deep Learning (LSTM) and deployed within a Jupyter Notebook environment.

## Features
- **Synthetic/Custom Data Pipeline**: Readily processes noisy sequential signals.
- **LSTM Architecture**: Built using TensorFlow/Keras with Dropout layers to prevent overfitting.
- **Multi-Step Forecasting**: Recursively forecasts multiple steps into the future.
- **Interactive Dashboard**: Uses `ipywidgets` to adjust forecast horizons on the fly.
